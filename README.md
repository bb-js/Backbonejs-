Backbone.js - [官网](http://backbonejs.org/)

    介绍:
    Backbone.js提供了一套web开发的框架，通过Models进行key-value绑定及自定义事件处理，
    通过Collections提供一套丰富的API用于枚举功能，通过Views来进行事件处理及与现有的
    Application通过RESTful JSON接口进行交互.它是基于jQuery和underscore的一个前端js框架。
